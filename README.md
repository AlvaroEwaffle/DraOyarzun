# Dra. Oyarzún - Landing Page

Landing page profesional para Dra. Carolina Oyarzún, especialista en estética facial en Talca, Chile.

## 🚀 Características

- **Framework**: Astro con Tailwind CSS
- **Diseño**: Mobile-first, responsive
- **Animaciones**: Suaves y profesionales
- **Performance**: Optimizado para carga rápida
- **SEO**: Meta tags optimizados
- **Conversión**: Enfocado en generar leads

## 🎨 Paleta de Colores

- **Main**: #FFC8DD (Rosa principal)
- **Hover**: #FFAFCC (Rosa hover)
- **Details**: #BDE0FE (Azul detalles)
- **Highlight**: #A2D2FF (Azul CTA)
- **Accent**: #CDB4DB (Púrpura secciones)

## 📱 Secciones

1. **Hero** - Presentación principal con CTA
2. **Benefits** - Beneficios clave de elegir a la Dra. Oyarzún
3. **Testimonials** - Testimonios de pacientes reales
4. **Services** - Servicios con sistema de tabs
5. **Offer** - Asesoría gratuita con formulario
6. **Guarantee** - Garantía de satisfacción
7. **FAQ** - Preguntas frecuentes
8. **CTA Final** - Llamada a la acción final
9. **Footer** - Información de contacto

## 🛠️ Instalación y Desarrollo

```bash
# Instalar dependencias
npm install

# Desarrollo
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview
```

## 📁 Estructura del Proyecto

```
src/
├── components/          # Componentes reutilizables
│   ├── ui/             # Componentes de UI base
│   ├── Header.astro    # Navegación
│   ├── Hero.astro      # Sección principal
│   ├── Benefits.astro  # Beneficios
│   ├── Testimonials.astro # Testimonios
│   ├── Services.astro  # Servicios
│   ├── Offer.astro     # Oferta
│   ├── Guarantee.astro # Garantía
│   ├── FAQ.astro       # Preguntas frecuentes
│   ├── CTA.astro       # Llamada a la acción
│   └── Footer.astro    # Pie de página
├── layouts/            # Layouts base
├── pages/              # Páginas
│   ├── index.astro     # Página principal
│   └── servicios/      # Páginas de servicios
├── styles/             # Estilos globales

```
